İleri Seviyede Java

Javada sıra ileri seviyede. Bu bölümü kısa tutacağım çünkü ben henüz ileri seviye java biliyorum diyemem. Peki ileri seviye java bildiğinizi nasıl anlarsınız? Yaptığınız projelerle mi? Aldığınız eğitimlerle mi?





Bu sorunun cevabı aslında hepimizin hayatında bir çok kez girdiği sınav. Bu sınav Oracle üzerinden yapılıyor ve eğer sınavı geçerseniz Oracle sertifikası alırsınzı ki bu çok iyi bir şey. Ancak sınava girmeden önce internetten kaynaklara da bakmanızı tavsiye ederim.

Yazdığım gibi kendim ileri seviye java biliyorum diyemem. Ancak araştırdığımda ileri seviye java bilenlerin en çok ilgilendiği şeylerden biri felsefe. Yani programlamaya felsefe gözüyle de bakıyorlar. Ayrıca soyutlama da onlar için çok önemli bir yer tutuyor.
