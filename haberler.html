HABERLER



Java Kullanan Teknoloji Şirketleri

Fortune dergisinin “En büyük 500 şirket” listesinde yer alan kurumların %90’ı Java yazılımı kullanıyor. Aslında bu veri Java’nın kurumsal bilişim alanındaki yerini çok iyi özetliyor. RedMonk’da analistlik yapan Stephen O’Grady, Java’nın uzun yıllar bizimle olmasını bekliyoruz diyor ve ekliyor: “Java, büyük veri, mobil, gibi kurumsal uygulamalarda çok önemli bir görev üstlenici. Her zaman popüler kalacak bir yazılım platformu gibi görünüyor.”

Java; Android uygulama geliştirmede en çok tercih edilen doğru yol

Paul Krill makalesinde, dünya genelinde bir numaralı mobil platformun iPhone ve iPad değil, Google’ın Android mobil platformu olduğunu belirtiyor ve Android için uygulama geliştirmede yazılımcıların ezici bir çoğunlukla Java becerilerini ve Dalvik VM kullandıklarını belirtiyor. Teknoloji araştırmaları şirketi Gartner’ın açıkladığı verilere göre geçtiğimiz yıl Android tabletler dünya çapında %62’lik bir satış oranı yakalarken, iOS %36’da kaldı. IDC’nin ikinci çeyrek verilerine göre ise Android telefon dünya çapında yaklaşık %82’lik bir pazar payına sahipken iOS 11.7’lik bir yüzdeye sahip. Bu tabi sadece mobil dünya için geçerli değil; akıllı televizyonlar, buzdolapları da Java’nın kabiliyetleri ile geliştirilebiliyor bugün.


Java değişime ayak uyduruyor ve gelişmeyi hiç bırakmıyor

Java 19 yıllık bir şirket. Çok yaşlı bir kurum olmamakla birlikte teknolojinin hızlı değişimine oranla 19 yıl pek çok şeyin geride bırakılıp değişimlerin yaşandığı anlamına geliyor. Platform kendine yeni özellikler katmaya devam ediyor. Bunlardan en güncel olanına örnek olarak Java 8’in standart versiyonunda yaptığı son iyileştirmeler söylenebilir. Java 9 ise 2016’da geliyor. Java EE (Enterprise Edition) 8’in de en yeni web standartlarına uyumlu, bulut destekli ve gelişme kolaylığı gibi özelliklere sahip olması bekleniyor.



Javanın Kurucusu: James GOSLING

James Arthur Gosling (d. 19 Mayıs 1955), Kanadalı bilgisayar bilimcisi. Java programlama dilinin yaratıcısı ve ana tasarımcısı olarak bilinmektedir.
Carnegie Mellon Üniversitesi'ndeki doktora çalışmaları sırasında çok işlemcili bir Unix sürümü geliştirmiş; daha sonra girdiği Sun Microsystems'ta ise birçok derleyici ve e-posta hizmeti yazılımına katkıda bulunmuştur.



Java İle Neler Yapılabilir?

Java sadece World Wide Web için sevimli, eğlenceli uygulamaları yazmak için değil. Java genel amaçlı, yüksek düzey programlama dili ve güçlü yazılım platformudur. Java diliyle projeler diğer programlama dillerine göre daha kolay, sağlıklı ve esnek şekilde yapılması mümkün olur.
Java ile herşey yapılabilir ve her yerde çalışır, peki bu ne anlama geliyor ? Örneğin Java ile Windowsda yazdığımız kod hiç değiştirmeden Linux'ta aynı şekilde cep telefonunda çalışır.Java ile Windows pogramları, bulaşık makinası yazılımı, uçak yazılımı gibi birçok yazılım yapılabilir.

Java, günümüzde 4,5 milyarı aşkın aygıta güç katmaktadır; bu aygıtlar arasında şunları sayabiliriz:


  • 800 milyon bilgisayar

  • 1,5 milyar cep telefonu ve diğer mobil aygıtlar

  • 2,2 milyar akıllı kart

  • Dekoderler, yazıcılar, Web kameraları, oyunlar, araçlar için yol bilgisayarları, sayısal loto bayileri, tıbbi cihazlar, otopark ödeme noktaları ve diğerleri.
